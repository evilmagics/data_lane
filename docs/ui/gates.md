# Gates Management

The Gates Management page allows users to manage gates.

## UI Features
- **Icon**: Represented by the `DoorOpen` icon.
- **List Gates**: View all gates with ID and Name.
- **Create Gate**:
    - Single: Add a gate manually (ID optional, Name required).
    - Batch: Add multiple gates via JSON `[{"id": 1, "name": "..."}]`.
- **Delete/Update**: Supports batch and single operations.

## Usage
Navigate to `/gates` via the Sidebar ("Gates" item with Door icon).
header displays "Gates" with the icon.
