# UI Project Configuration

## Git Configuration

The UI project (`/ui`) uses a specific `.gitignore` to ensure that dependencies, build artifacts, and local environment secrets are not committed to source control.

### Ignored Files
- **Dependencies**: `node_modules`
- **Next.js Artifacts**: `.next/`, `out/`
- **Build Output**: `dist`, `build`
- **Environment**: `.env`, `.env*.local`
- **Editor/OS**: `.vscode`, `.idea`, `.DS_Store`, `Thumbs.db`

### Notes
- `bun.lock` (text format) IS committed to ensure consistent dependency resolution.
- `next-env.d.ts` is ignored as it is auto-generated by Next.js.
